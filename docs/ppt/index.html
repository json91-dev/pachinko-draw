Pachinko Draw – 사용된 라이브러리

Runtime Dependencies

matter-js2D 물리 엔진. 충돌, 중력, 강체 시뮬레이션
next16.1.6 — App Router, 서버 컴포넌트 기반 프레임워크
react19.2.3 — UI 렌더링, 상태 관리, Hooks
react-dom19.2.3 — DOM 바인딩

Dev Dependencies

typescript타입 안전성, 컴파일 타임 에러 검출
tailwindcss유틸리티 CSS 프레임워크 (v4)
eslint코드 품질/스타일 린터

왜 이 조합인가?

  • Matter.js — 가볍고 브라우저 네이티브, 별도 WASM 불필요
  • Next.js 16 — dynamic import로 PachinkoBoard 지연 로딩
  • React 19 — useCallback/useRef로 게임 루프와 상태 분리
  • Canvas 2D — WebGL 없이도 수백 개 공 60fps 렌더링
외부 의존성이 matter-js 단 1개인 초경량 구조.
나머지는 모두 Next.js/React 생태계 기본 도구.

Matter.js – 사용된 기능

2D 물리 엔진으로 핀볼 시뮬레이션의 핵심을 담당

Engine.create / Engine.update
중력(0.96) 설정 후 매 프레임 물리 시뮬레이션 갱신.
피날레 모드에서 timeScale = 0.25로 슬로우모션 연출.
Bodies.circle / Bodies.rectangle
공(circle, r=10), 핀(circle, r=10, isStatic),
벽(rectangle, isStatic)을 생성해 월드에 배치.
Body.setVelocity / setPosition
대포 발사 시 각도별 초기 속도 부여.
블랙홀 인력은 setVelocity로 방향 조향.
Body.setAngle / Body.rotate
물레방아 날개 회전 애니메이션.
중심축 기준 setPosition + setAngle 갱신.
World.add / World.remove
공 발사 시 add, 홀 진입/화면 밖 시 remove.
블랙홀 성장 시 센서 교체(remove → 재생성 → add).
collisionFilter
홀은 isSensor: true로 통과 감지만.
블랙홀 근처 공 간 충돌 비활성화(group = -1).

승리 조건

  • 조기 판정 — 1위 점수 > 2위 + 남은 공 수
  • 전체 소진 — 모든 공 소진 시 최고 점수자
  • 타임아웃 — 피날레 15초 초과 시 강제 종료

주요 연출

  • 블랙홀 — 반경 점진 성장, 인력 lerp 유도
  • 피날레 — timeScale 0.25, 카메라 1.8x 줌
  • 핀 소멸 — 블랙홀 주변 핀 순차 제거
  • 우승 화면 — glow 텍스트 + 파티클 폭죽
